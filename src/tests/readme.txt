Open terminal from /test and use command

    <build>/crypto/fift -I <source>/crypto/fift/lib -s <name>.fif

for each file. If the test passed the last line will be "ok!".

Folder /fc contains funC source files.
Folder /fif contains fift tvm assembler code files for each file from folder /fc.
Folder /wallets contains .pk files generating during tests.
